HTML,CSS与JavaScript之间的联系与区别

网页必备技术，结构：HTML，样式：CSS，动作：JavaScript

HTML 是用来描述网页的一种超文本标记语言 ，是一套标记符号或者代码集，它们标识了网页的结构元素，如段落，标题和列表，以及超链接等。 因此，HTML主要用来描述网页结构。除此之外还可以用HTML在网页上放置多媒体或对表单进行描述。

CSS它是一种用来表现HTML等文件的计算机语言，可以将排版样式（字号，字体），颜色和页面布局指令应用于网页。

JavaScript是作为一种嵌入HTML文档的，动态的，基于对象的脚本语言。可以控制HTML中的元素，可以添加，删除，修改等。为网页浏览效果服务。
